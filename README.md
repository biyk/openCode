# PyQt6 Voice → Summary → Polite Email (OpenRouter + Vosk) — Mini Demo

**Что это:** простое офлайн STT + онлайн LLM демо. Микрофон → живой текст (Vosk) → 5‑пунктный конспект → короткое вежливое письмо, при желании перевод на английский.  
**Бесплатные компоненты:** Vosk (офлайн ASR), PyQt6, sounddevice, requests. LLM доступ через OpenRouter.

## Быстрый старт

1) Установите зависимости в виртуальном окружении (Python 3.10+):

```bash
pip install -r requirements.txt
```

2) Ключ OpenRouter: в `.env` уже положен ваш ключ. **В реальном проекте так не делайте**, храните секреты безопасно.

3) Запуск:
```bash
python main.py
```

4) При первом запуске нужная Vosk‑модель(и) скачается(т) ~50–60 МБ каждая в папку `models/`:
   - Русская: `vosk-model-small-ru-0.22`
   - Английская (US): `vosk-model-small-en-us-0.15`

5) В окне приложения:
   - Выберите язык распознавания (RU/EN).
   - Нажмите **Старт** — идёт живое распознавание микрофона.
   - **Стоп** — остановка.
   - **Суммаризировать** — отправка текста в LLM через OpenRouter, генерация:
     - 5 пунктов‑выжимки
     - Тема письма
     - Вежливое письмо на русском
     - Если включена галочка «Перевести на английский», дополнительно будет создан английский вариант письма.

## Требования/заметки

- Mac/Windows/Linux. На macOS может потребоваться разрешение на доступ к микрофону.
- Vosk работает офлайн и быстро на CPU. Точность «small» моделей приемлемая для демо, для продакшена используйте «big» модели.
- Звук берётся с системного устройства ввода (1 канал, 16 kHz).

## Безопасность ключей

Здесь `.env` добавлен ради простоты демо. В бою храните ключ в переменных окружения или секрет‑хранилищах. Не коммитьте `.env` в репозиторий.
